# Creature Simulator

A simple 2D walk simulator built with C++ and raylib. Nodes spawn and move in random directions.

![GIF](gameplay.gif)

## Features

* Spawn multiple nodes with randomized directional movement.
* Adjustable speeds for simulation testing.

## Getting Started

### Dependencies

* Windows 11
* C++ Compiler (e.g., MinGW-w64 GCC)
* [raylib](https://www.raylib.com/)
* VSCode (with C/C++ extension)

### Installing & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/asm-enjoyer/creature_sim.git
   ```
2. Ensure your C++ compiler is added to your system's PATH.
3. For detailed instructions on setting up raylib and the required environment, please refer to [this setup guide repository's](https://github.com/educ8s/Raylib-CPP-Starter-Template-for-VSCODE-V2) video tutorial.

### Executing Program

1. Open the cloned repository folder in VSCode.
2. Navigate to the "Run and Debug" panel.
3. Select the provided 'Run' or 'Debug' configuration from the dropdown.
4. Start debugging (F5).
